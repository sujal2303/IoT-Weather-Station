# IoT Weather Station

## Project Overview

The IoT Weather Station is a smart monitoring system that collects environmental data such as temperature, humidity, and atmospheric conditions using IoT sensors.

The collected data can be displayed on a dashboard and monitored remotely.

## Features

- Real-time temperature monitoring
- Humidity measurement
- Weather data collection
- Remote monitoring
- IoT-based automation

## Components Used

- ESP8266 / ESP32
- DHT11 Sensor
- Wi-Fi Module
- Cloud Dashboard

## Technologies

- Python
- IoT
- Embedded Systems

## Future Enhancements

- Rain Detection
- Air Quality Monitoring
- Mobile Application Integration
- Weather Forecast Prediction
